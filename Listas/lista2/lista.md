### **Exercício 1: Acessando o Primeiro Elemento**

Crie uma lista chamada `cores` com os elementos `"vermelho"`, `"azul"`, `"verde"`, `"amarelo"`. Acesse e imprima o
primeiro elemento da lista.

### **Exercício 2: Acessando o Último Elemento**

Usando a lista `cores` do exercício anterior, acesse e imprima o último elemento da lista utilizando um índice negativo.

### **Exercício 3: Modificando um Elemento**

Crie uma lista chamada `numeros` com os valores `[5, 10, 15, 20, 25]`. Modifique o valor do terceiro elemento da lista
para `30` e imprima a lista atualizada.

### **Exercício 4: Acessando um Índice Fora do Alcance**

Crie uma lista chamada `animais` com os elementos `"gato"`, `"cachorro"`, `"papagaio"`. Tente acessar o elemento no
índice `5` e observe o erro gerado.

### **Exercício 5: Fatiamento Simples**

Usando a lista `numeros` criada no exercício 3, faça um fatiamento para obter os três primeiros elementos da lista e
imprima o resultado.

### **Exercício 6: Fatiamento com Índices Negativos**

Usando a mesma lista `numeros`, faça um fatiamento para obter os dois últimos elementos usando índices negativos e
imprima o resultado.

### **Exercício 7: Iterando com Índices**

Crie uma lista chamada `frutas` com os elementos `"maçã"`, `"banana"`, `"laranja"`, `"uva"`. Use um loop `for` para
imprimir cada elemento junto com seu índice correspondente.

### **Exercício 8: Somando Dois Elementos Usando Índices**

Crie uma lista chamada `valores` com os elementos `[100, 200, 300, 400, 500]`. Some o segundo e o quarto elementos da
lista e imprima o resultado.

### **Exercício 9: Modificando Vários Elementos com Fatiamento**

Usando a lista `valores` do exercício anterior, altere os dois últimos elementos para `[600, 700]` usando fatiamento e
imprima a lista atualizada.

### **Exercício 10: Verificando a Ocorrência de um Índice**

Crie uma lista chamada `idades` com os valores `[20, 30, 40, 50, 60]`. Verifique se existe um valor no índice `3` e
imprima o valor encontrado.

Esses exercícios irão reforçar suas habilidades em acessar, modificar e manipular elementos de listas usando índices e
fatiamento em Python.
